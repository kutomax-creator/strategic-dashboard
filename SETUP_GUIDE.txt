================================================================
  Strategic Dashboard - セットアップ手順書
================================================================


■ 概要

  macOS で開発 → pip が使える Windows PC で .exe ビルド
  → 会社 PC には .exe フォルダをコピーするだけ

  会社PCに Python や pip は不要です。


================================================================
  全体の流れ
================================================================

  [macOS]        build_package.sh 実行
                     ↓
  [自宅WinPC]    StrategicDashboard/ をコピー
                 build_exe.bat 実行（.exe 生成）
                     ↓
  [会社WinPC]    dist\StrategicDashboard\ をコピー
                 StrategicDashboard.exe をダブルクリック


================================================================
  STEP 1: macOS で配布フォルダを作る
================================================================

  ターミナルで以下を実行:

    ./build_package.sh

  → StrategicDashboard/ フォルダが生成される


================================================================
  STEP 2: 自宅 Windows PC で .exe をビルド
================================================================

  前提:
    - Python 3.10 以上がインストール済み
    - pip が使える（インターネット接続あり）

  手順:
    1) StrategicDashboard/ フォルダを自宅 Windows PC にコピー

    2) フォルダ内の build_exe.bat をダブルクリック

    3) 自動で以下が行われる:
       - pip install（必要パッケージ）
       - PyInstaller で .exe ビルド
       ※ 初回は数分かかります

    4) 完了すると dist\StrategicDashboard\ フォルダが生成される
       中に StrategicDashboard.exe がある


================================================================
  STEP 3: 会社 PC で使う
================================================================

  1) dist\StrategicDashboard\ フォルダを
     会社PCの任意の場所にコピー

     例: デスクトップ\StrategicDashboard\

  2) StrategicDashboard.exe をダブルクリック

  3) 数秒後にウィンドウが開き、ダッシュボードが表示される
     ※ ブラウザは使いません（ネイティブウィンドウ）

  4) 閉じるときはウィンドウの × ボタンを押す


================================================================
  トラブルシューティング
================================================================

  Q: build_exe.bat でエラーが出る
  A: Python のバージョンを確認（3.10以上が必要）
     python --version で確認できる

  Q: .exe を起動しても何も表示されない
  A: 少し待つ（初回は起動に10秒程度かかる場合がある）
     それでもダメなら、コマンドプロンプトから
     StrategicDashboard.exe を実行し、エラーを確認する

  Q: ウィンドウは開くがデータが表示されない
  A: 会社PCからインターネットに接続できるか確認する
     （株価・ニュース取得にネット接続が必要）

  Q: AI分析が動かない
  A: Anthropic API キーが有効か確認する
     （config.py 内の ANTHROPIC_API_KEY）

  Q: アプリを更新したい
  A: STEP 1 → STEP 2 → STEP 3 を再度実行する
